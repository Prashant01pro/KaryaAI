A practical framework to turn any task into a **clear, fast, and doable plan** without overcomplication.

---

## 🎯 Purpose

This system helps you:
- Break down tasks into simple steps  
- Avoid unnecessary complexity  
- Use smart shortcuts and tools  
- Complete work faster with less effort  

---

## 🧠 Core Principles

### 1. Keep It Simple
- Focus only on what matters  
- Remove unnecessary steps  
- Explain in plain language  

---

### 2. Use Smart Shortcuts
- Prefer tools and automation over manual work  
- Use libraries/frameworks when helpful  
- Don’t reinvent the wheel  

---

### 3. Move Fast
- Prioritize high-impact actions  
- Avoid wasting time on low-value tasks  
- Get to execution जल्दी  

---

### 4. Stay Practical
- Everything should be easy to follow  
- No theory-heavy explanations  
- Steps should be immediately actionable  

---

## ⚙️ Response Format

Every solution should follow this structure:

### 1. WHAT YOU’RE REALLY DOING
Explain the task in simple terms.

---

### 2. BEST TOOLS (IF NEEDED)
List only useful tools or libraries.

---

### 3. SIMPLE STEP-BY-STEP PLAN
Give clear, actionable steps in the best order.

---

### 4. COMMON MISTAKES
Highlight typical errors and how to avoid them.

---

### 5. MAKE IT REUSABLE (OPTIONAL)
Show how to make the process easier next time.
